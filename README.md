This repository is a comprehensive collection of Machine Learning projects, implementing both **Supervised** and **Unsupervised Learning** techniques. It covers the entire ML pipeline: from exploratory data analysis (EDA) and feature engineering to model training, hyperparameter tuning, and performance evaluation.

---

## 📂 Repository Structure

The projects are organized by learning type and algorithm category:

### 📈 Regression
*Predicting continuous values (e.g., house prices, stock trends).*
- **Linear & Polynomial Regression**
- **Ridge & Lasso Regression** (Regularization)

### 📊 Classification
*Predicting discrete categories (e.g., spam detection, disease diagnosis).*
- **Logistic Regression**
- **Decision Trees & Random Forest**
- **Support Vector Machines (SVM)**
- **K-Nearest Neighbors (KNN)**

### 🧩 Clustering
*Grouping data points without predefined labels.*
- **K-Means Clustering**
- **DBSCAN**
- **Hierarchical Clustering**

### 🔍 Association Rule Learning
- **Apriori Algorithm**
- **Eclat**

### 🤖 Recommendation Systems
- **Content-Based Filtering**
- **Collaborative Filtering**
